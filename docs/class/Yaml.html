Class: Yaml

Defined in: node_modules/yamljs/src/Yaml.coffee

Overview

Yaml offers convenience methods to load and dump YAML.

Class Method Details

. (Object) parse(input, exceptionOnInvalidType = false, objectDecoder = null)

Parses YAML into a JavaScript object.

The parse method, when supplied with a YAML string, will do its best to convert YAML in a file into a JavaScript object.

Usage: myObject = Yaml.parse('some: yaml'); console.log(myObject);

Parameters:

  • input ( String ) — A string containing YAML
  • exceptionOnInvalidType ( Boolean ) — true if an exception must be thrown on invalid types, false otherwise
  • objectDecoder ( Function ) — A function to deserialize custom objects, null otherwise

Throws:

  • ( ParseException ) — If the YAML is not valid

Returns:

  • ( Object ) — The YAML converted to a JavaScript object

. (Object) parseFile(path, callback = null, exceptionOnInvalidType = false, objectDecoder = null)

Parses YAML from file path into a JavaScript object.

The parseFile method, when supplied with a YAML file, will do its best to convert YAML in a file into a JavaScript object.

Usage: myObject = Yaml.parseFile('config.yml'); console.log(myObject);

Parameters:

  • path ( String ) — A file path pointing to a valid YAML file
  • exceptionOnInvalidType ( Boolean ) — true if an exception must be thrown on invalid types, false otherwise
  • objectDecoder ( Function ) — A function to deserialize custom objects, null otherwise

Throws:

  • ( ParseException ) — If the YAML is not valid

Returns:

  • ( Object ) — The YAML converted to a JavaScript object or null if the file doesn't exist.

. (String) dump(input, inline = 2, indent = 4, exceptionOnInvalidType = false, objectEncoder = null)

Dumps a JavaScript object to a YAML string.

The dump method, when supplied with an object, will do its best to convert the object into friendly YAML.

Parameters:

  • input ( Object ) — JavaScript object
  • inline ( Integer ) — The level where you switch to inline YAML
  • indent ( Integer ) — The amount of spaces to use for indentation of nested nodes.
  • exceptionOnInvalidType ( Boolean ) — true if an exception must be thrown on invalid types (a JavaScript resource or object), false otherwise
  • objectEncoder ( Function ) — A function to serialize custom objects, null otherwise

Returns:

  • ( String ) — A YAML string representing the original JavaScript object
