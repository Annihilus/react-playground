Class: Pattern

Defined in: node_modules/yamljs/src/Pattern.coffee

Overview

Pattern is a zero-conflict wrapper extending RegExp features in order to make YAML parsing regex more expressive.

Property Summary

(RegExp) regex
The RegExp instance
(String) rawRegex
The raw regex string
(String) cleanedRegex
The cleaned regex string (used to create the RegExp instance)
(Object) mapping
The dictionary mapping names to capturing bracket numbers

Instance Method Summary

  • # (Array) exec(str) Executes the pattern's regex and returns the matching values
  • # (Boolean) test(str) Tests the pattern's regex
  • # (String) replace(str, replacement) Replaces occurences matching with the pattern's regex with replacement
  • # (Array) replaceAll(str, replacement, limit = 0) Replaces occurences matching with the pattern's regex with replacement and get both the replaced string and the number of replaced occurences in the string.

# (Array) replaceAll(str, replacement, limit = 0)

Replaces occurences matching with the pattern's regex with replacement and get both the replaced string and the number of replaced occurences in the string.

Parameters:

  • str ( String ) — The source string to perform replacements
  • replacement ( String ) — The string to use in place of each replaced occurence.
  • limit ( Integer ) — The maximum number of occurences to replace (0 means infinite number of occurences)

Returns:

  • ( Array ) — A destructurable array containing the replaced string and the number of replaced occurences. For instance: ["my replaced string", 2]
