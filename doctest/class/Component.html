Class: Component

Defined in: src/scripts/core/component.coffee

Overview

Base component class, the prototype of all project components.

Examples:

Component creation 

class Dropdown extends Component
  locals: ->
    msg: "hello"

Component instantiation 

Dropdown = require "dropdown"
Dropdown

  # Properties
  text: "my button"
  onClick: ->
    alert("click")

  # Children
  [
    Link to: "users"   # children also may be passed
    Link to: "events"  # via properties with `children` key.
  ]                    # If children were passed in 3-rd param -
                       # props.children will be ignored
